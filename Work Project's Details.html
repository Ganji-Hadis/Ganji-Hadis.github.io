  • Designing a board to support Power on Ethernet (PoE) for setting up surveillance cameras and providing information about the product's operational status.

    Details: In this project, we begin by receiving power and network data through a network socket (WAN port). Subsequently, we carefully separate these inputs to perform two essential tasks. Firstly, we directly supply power to the second network socket (LAN) to activate the camera. Secondly, we step down the 12 V input voltage to 5 V, enabling its utilization for the board's internal components. Additionally, we establish connections between the data received from the LAN socket and other sections of the board. This process involves receiving encrypted data, which then gets transferred to the global network for onward transmission to the organization. For user convenience and operational debugging, we incorporate 5 LEDs into the system. These LEDs are effectively interfaced with the processor's I2C pins via a PCF8574ap module, allowing them to convey essential information during operation.

    Key Instruments: Altium Designer Software, RJ45 Sockets, LEDs, LM2596 Module, Nano pi r2s Board, 2200W Capacitor, PCF8574ap IC, Switch Button, 220 & 10K Resistances,BC337 Transistors,Fuse,SMBJ5.0A TVS,IRF630 Transistor
